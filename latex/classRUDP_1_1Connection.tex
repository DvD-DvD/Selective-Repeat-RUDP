\doxysubsection{Detailed Description}
Used to create a connection, transfer data between server and client. 

Creates a udp connection with reliability between the server and the client.

Starts with a 3 way handshake and then the file trnasfer starts It checks for acknowledgement for each packet sent After a time of 3 seconds, if an ack for a packet is still not received, then the packet is sent again This can take place for a maximum of 5 times or maximum time of 30s (default values), whichever comes first
